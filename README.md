# Open-Meteo Weather Driver for Hubitat Elevation

This is a Hubitat Elevation driver for retrieving weather forecast data from the [Open-Meteo API](https://open-meteo.com/). It provides a 7-day weather forecast and allows for extensive customization of the data you wish to retrieve.

## Features

*   **7-Day Forecast:** Retrieves a full 7-day weather forecast.
*   **Highly Customizable:** Allows you to select which daily weather variables to fetch from the API.
*   **User-Configurable Units:** Supports both Fahrenheit/Celsius for temperature and mph/kmh for wind speed.
*   **Standard Capabilities:** Implements standard Hubitat capabilities for `TemperatureMeasurement` and `UltravioletIndex` for the current day, ensuring compatibility with Hubitat apps.
*   **Scheduled Polling:** Automatically refreshes weather data at a user-defined interval.
*   **Robust Error Handling:** Includes error handling for API requests and data parsing.

## Installation

1.  **Open the Hubitat Elevation UI:** Navigate to your Hubitat hub's web interface.
2.  **Go to the Drivers Code Section:** In the left-hand menu, click on "Drivers Code".
3.  **Add a New Driver:** Click the "+New Driver" button in the top-right corner.
4.  **Copy the Driver Code:** Copy the entire contents of the `OpenMeteo-Weather-Driver.groovy` file from this repository.
5.  **Paste the Code:** Paste the code into the text editor in the Hubitat UI.
6.  **Save the Driver:** Click the "Save" button in the top-right corner.
7.  **Create a Virtual Device:**
    *   Go to "Devices" in the left-hand menu.
    *   Click "+Add Device".
    *   Click "Virtual".
    *   Enter a "Device Name" (e.g., "Open-Meteo Weather").
    *   Under "Type", select "Open-Meteo Weather Driver" from the dropdown list.
    *   Click "Save Device".

## Configuration

Once the virtual device is created, you can configure the driver in the "Preferences" section of the device's page.

### API Configuration

*   **Latitude:** The latitude for the weather forecast.
*   **Longitude:** The longitude for the weather forecast.
*   **Timezone:** The timezone for the location (e.g., "America/Chicago").
*   **Poll Interval:** How often to refresh the weather data.
*   **Temperature Unit:** Choose between "Fahrenheit" and "Celsius".
*   **Wind Speed Unit:** Choose between "mph" and "kmh".

### Daily Weather Variables

This section allows you to select which weather variables you want to retrieve from the Open-Meteo API. Enable the toggle for each variable you wish to see.
